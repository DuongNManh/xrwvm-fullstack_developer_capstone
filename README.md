# xrwvm-fullstack_developer_capstone

## Project Name: Fullstack Developer Capstone Project

This repository contains the source code for the **Fullstack Developer Capstone Project**, part of the IBM Fullstack Developer specialization.  
It is a complete web application that integrates a **Django backend**, a **React frontend**, and a **Node.js microservice** for dealership and review management, with MongoDB as the database.

### Features
- User authentication (login, logout, registration)
- Dealership listing and filtering by state
- Dealer detail pages with customer reviews
- Sentiment analysis for reviews using a microservice
- Ability to post new reviews when logged in
- Dockerized setup with MongoDB and Django for easy deployment

### Tech Stack
- **Frontend:** React, CSS
- **Backend:** Django
- **Microservice:** Node.js + Express + Mongoose
- **Database:** MongoDB
- **Deployment:** Docker & Docker Compose
